# Bokkna - Application de Vote Électronique

Bokkna est une application web de vote conçue pour moderniser, sécuriser et simplifier le processus de vote. Ce projet a été développé dans le cadre de notre formation à l'ISEPAT.

## 🚀 Fonctionnalités Principales
* **Gestion des scrutins :** Création et configuration des votes.
* **Authentification sécurisée :** Pour s'assurer que chaque utilisateur ne vote qu'une seule fois.
* **Vote en temps réel :** Prise en compte instantanée des voix.
* **Interface Backend :** API REST développée pour une communication fluide des données.
